# Лабораторная работа №1 "Реализация арифметических операций с числами в различных двоичных кодах"

## Цель работы

Разработать программный комплекс для выполнения арифметических операций с числами в различных двоичных представлениях:

- Прямой, обратный и дополнительный коды для целых чисел
- Стандарт IEEE-754-2008 для чисел с плавающей точкой
- Двоично-десятичный код (8421 BCD)

**Основное требование:** все операции должны быть реализованы без использования встроенных функций преобразования, с хранением чисел в виде массивов битов размером 32.

---

## Теоретическая часть

### Прямой код
В прямом коде старший бит отводится под знак числа (0 - положительное, 1 - отрицательное), остальные биты хранят модуль числа.

### Обратный код
Для положительных чисел совпадает с прямым кодом. Для отрицательных чисел все биты кроме знакового инвертируются.

### Дополнительный код
Для положительных чисел совпадает с прямым кодом. Для отрицательных чисел получается из обратного кода добавлением 1.

### IEEE-754 (32 бита)
Число представляется тремя полями:
- **Знак (1 бит):** 0 - положительное, 1 - отрицательное
- **Экспонента (8 бит):** хранится со смещением 127
- **Мантисса (23 бита):** нормализованная дробная часть

### BCD (8421)
Каждая десятичная цифра кодируется 4 битами. В проекте используется 8 цифр = 32 бита.

## Описание классов и методов

### 1. Класс `BinaryRepresentation` (binary_representation.py)

Класс для представления чисел в двоичном коде (32 бита).

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `decimal_to_binary()` | Преобразование десятичного числа в двоичный массив | `num` - целое число | Массив из 32 битов |
| `binary_to_decimal()` | Преобразование двоичного массива в десятичное число | `binary` - массив битов | Десятичное число |
| `direct_code()` | Получение прямого кода числа | `num` - целое число | Массив битов в прямом коде |
| `reverse_code()` | Получение обратного кода числа | `num` - целое число | Массив битов в обратном коде |
| `additional_code()` | Получение дополнительного кода числа | `num` - целое число | Массив битов в дополнительном коде |
| `additional_to_decimal()` | Преобразование дополнительного кода в десятичное число | `additional` - массив битов | Десятичное число |
| `binary_add()` | Сложение двух двоичных массивов | `a`, `b` - массивы битов | Массив-результат |
| `add_additional()` | Сложение в дополнительном коде | `num1`, `num2` - числа | Массив-результат |
| `subtract_additional()` | Вычитание в дополнительном коде | `num1`, `num2` - числа | Массив-результат |
| `negate_additional()` | Отрицание числа в дополнительном коде | `num` - число | Массив-результат |
| `multiply_direct()` | Умножение в прямом коде | `num1`, `num2` - числа | Массив-результат |
| `divide_direct()` | Деление в прямом коде | `num1`, `num2`, `precision` | Результат и прямой код |
| `binary_to_string()` | Преобразование двоичного массива в строку | `binary` - массив битов | Строка из 0 и 1 |

### 2. Класс `FloatIEEE754` (float_ieee754.py)

Класс для работы с числами с плавающей точкой по стандарту IEEE 754-2008 (32 бита).

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `decimal_to_float32()` | Преобразование десятичного числа в формат IEEE 754 | `num` - число с плавающей точкой | Массив из 32 битов |
| `float32_to_decimal()` | Преобразование из IEEE 754 в десятичное число | `binary` - массив битов | Число с плавающей точкой |
| `add_float()` | Сложение двух чисел | `a`, `b` - числа | Массив-результат и десятичный результат |
| `subtract_float()` | Вычитание двух чисел | `a`, `b` - числа | Массив-результат и десятичный результат |
| `multiply_float()` | Умножение двух чисел | `a`, `b` - числа | Массив-результат и десятичный результат |
| `divide_float()` | Деление двух чисел | `a`, `b` - числа | Массив-результат и десятичный результат |
| `float_to_string()` | Преобразование в строку с разбором полей | `binary` - массив битов | Строка вида "Знак: X | Экспонента: XXXXXXXX | Мантисса: XXX..." |

### 3. Класс `BCD8421` (bcd8421.py)

Класс для работы с двоично-десятичным кодом 8421 (32 бита = 8 цифр).

| Метод | Назначение | Входные данные | Выходные данные |
|-------|------------|----------------|-----------------|
| `decimal_to_bcd()` | Преобразование десятичного числа в BCD | `num` - неотрицательное число | Массив из 32 битов |
| `bcd_to_decimal()` | Преобразование BCD в десятичное число | `bcd` - массив битов | Десятичное число |
| `bcd_add()` | Сложение двух чисел в BCD | `num1`, `num2` - числа | Массив-результат |
| `bcd_to_string()` | Преобразование BCD в строку | `bcd` - массив битов | Строка вида "0001 0010 0011..." |

Главное меню:

1)Преобразование числа в прямой, обратный и дополнительный коды

2)Сложение двух чисел в дополнительном коде

3)Вычитание двух чисел в дополнительном коде

4)Умножение двух чисел в прямом коде

5)Деление двух чисел в прямом коде (с точностью до 5 знаков)

6)Операции с числами с плавающей точкой (IEEE 754-2008)

7)Сложение чисел в двоично-десятичном коде 8421 (BCD)

8)Выход
